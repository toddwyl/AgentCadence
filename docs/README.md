# Docs & media

Add demo and screenshot assets here; the root **README.md** references them.

## Recommended files

| File | Suggested content |
|------|-------------------|
| `demo.gif` | Screen recording: create pipeline → configure steps → Run → Run Monitor (keep under ~5 MB if possible). |
| `pipeline-editor.png` | Main editor with stages/steps. |
| `run-monitor.png` | Run Monitor with Running / History. |
| `settings-cli.png` | Settings with CLI detection. |

## Image hosting

Use **relative** paths in Markdown for files in this repo, or absolute **raw GitHub** URLs, for example:

```text
https://github.com/toddwyl/AgentCadence/raw/main/docs/demo.gif
```

After adding `demo.gif`, uncomment the `<img>` block in the root `README.md` under **Screenshots & demo**.
